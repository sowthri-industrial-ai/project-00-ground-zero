# ADR-0019 · ALLaM for regional AI landscape signal

**Status**: Accepted · 2026-04-24

## Context
Portfolio targets GenAI Architect role in KSA. SDAIA's ALLaM-2-7b-instruct is the regional Arabic-English LLM reviewers expect awareness of.

## Decision
Thin demo in Ground Zero: Foundry-deployed ALLaM managed endpoint · ephemeral per ADR-0013. One bilingual query path. Content Safety + groundedness work on Arabic content.

## Consequences
- Demonstrates deployment patterns for Arabic-capable models
- Does NOT commit Ground Zero to Arabic-primary use cases
- Flagship project (Project 3) is correct placement for full integration
